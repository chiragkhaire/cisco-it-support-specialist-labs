# Hardware Support

## Responsibilities

- Diagnose hardware failures
- Replace faulty components
- Upgrade existing hardware
- Install peripherals
- Perform preventive maintenance
- Verify system functionality after repairs

---

## Common Components

- CPU
- Motherboard
- RAM
- SSD / HDD
- Power Supply Unit (PSU)
- GPU
- Cooling System
- CMOS Battery

---

## Common Hardware Issues

- No power
- No display
- Overheating
- Slow performance
- Storage failure
- Faulty RAM
- Fan noise
- Peripheral failure

---

## Safety

- Shut down the computer
- Disconnect power
- Wear an anti-static wrist strap when available
- Handle components by the edges
- Keep liquids away from electronics
