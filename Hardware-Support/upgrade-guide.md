# Hardware Upgrade Guide

## RAM Upgrade

- Verify motherboard compatibility
- Match memory type (DDR4/DDR5)
- Install in recommended slots
- Verify detection in BIOS

---

## SSD Upgrade

Steps

1. Backup important data
2. Install SSD
3. Clone or reinstall operating system
4. Verify boot order
5. Confirm storage detection

---

## CPU Upgrade

- Verify motherboard compatibility
- Update BIOS if required
- Apply fresh thermal paste
- Install cooler correctly
- Monitor temperatures after installation

---

## Best Practices

- Ground yourself before touching components
- Keep screws organized
- Verify compatibility before purchasing hardware
