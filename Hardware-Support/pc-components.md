# PC Components Overview

| Component | Function |
|-----------|----------|
| CPU | Executes instructions |
| Motherboard | Connects all hardware |
| RAM | Temporary memory |
| SSD | Fast permanent storage |
| HDD | Mechanical storage |
| GPU | Graphics processing |
| PSU | Supplies power |
| NIC | Network connectivity |
| CMOS Battery | Maintains BIOS settings |

---

## BIOS vs UEFI

| BIOS | UEFI |
|------|------|
| Legacy firmware | Modern firmware |
| Smaller disk support | Large disk support |
| Slower boot | Faster boot |
| Limited features | Secure Boot support |

---

## POST

Power-On Self-Test (POST) checks hardware before the operating system starts.

If POST fails, beep codes or diagnostic LEDs may indicate the faulty component.
