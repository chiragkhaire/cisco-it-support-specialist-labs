# Ticket Lifecycle

```text
User Reports Issue
        │
        ▼
Ticket Created
        │
        ▼
Initial Assessment
        │
        ▼
Assign Priority
        │
        ▼
Troubleshooting
        │
        ▼
Resolved?
   │          │
 Yes         No
 │            │
 ▼            ▼
Close     Escalate
Ticket    to Higher Tier
```

## Ticket Status

- New
- Open
- In Progress
- Waiting for User
- Escalated
- Resolved
- Closed

## Priority Levels

| Priority | Description |
|----------|-------------|
| P1 | Critical |
| P2 | High |
| P3 | Medium |
| P4 | Low |

## SLA

Service Level Agreement (SLA) defines the expected response and resolution time for support requests.
