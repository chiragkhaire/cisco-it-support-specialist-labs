# Basic Incident Response

## Incident Lifecycle

```text
Identify
   ↓
Contain
   ↓
Investigate
   ↓
Eradicate
   ↓
Recover
   ↓
Document
```

---

## Example

### Incident

Employee reports ransomware warning.

### Actions

- Disconnect device from network
- Notify security team
- Preserve evidence
- Identify affected systems
- Restore from backup
- Verify system integrity

---

## Documentation

Record:

- Time of incident
- User affected
- Systems involved
- Actions taken
- Resolution
- Lessons learned
