# Security & Connectivity Support

## Responsibilities

- Troubleshoot network connectivity issues
- Configure basic network settings
- Apply security best practices
- Support VPN and remote access
- Maintain endpoint security
- Respond to security incidents

---

## Networking Components

- Router
- Switch
- Access Point
- Firewall
- Modem
- DHCP Server
- DNS Server

---

## Common Connectivity Issues

- No Internet
- Slow Network
- DNS Resolution Failure
- IP Address Conflict
- VPN Connection Failure
- Weak Wi-Fi Signal

---

## Security Principles

- Least Privilege
- Defense in Depth
- Confidentiality
- Integrity
- Availability

---

## Common Threats

- Malware
- Ransomware
- Phishing
- Spyware
- Social Engineering
- Password Attacks

---

## Best Practices

- Use strong passwords
- Enable MFA
- Keep systems updated
- Encrypt sensitive data
- Backup important files
- Lock devices when unattended
