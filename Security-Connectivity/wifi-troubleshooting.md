# Wi-Fi Troubleshooting

## Symptoms

- No Internet
- Slow Speeds
- Frequent Disconnects
- Weak Signal
- Authentication Failed

---

## Troubleshooting Steps

1. Verify Wi-Fi is enabled
2. Check Airplane Mode
3. Restart device
4. Restart router
5. Forget and reconnect to network
6. Check IP configuration
7. Test another device
8. Move closer to the access point

---

## Useful Commands (Windows)

```cmd
ipconfig
ping
netsh wlan show profiles
netsh wlan show interfaces
```

---

## Common Causes

- Weak signal
- Incorrect password
- DHCP issues
- Router failure
- ISP outage
- Driver issues
