| Layer | Function |
|--------|----------|
| 7 | Application |
| 6 | Presentation |
| 5 | Session |
| 4 | Transport |
| 3 | Network |
| 2 | Data Link |
| 1 | Physical |

## TCP/IP Model

| Layer | Purpose |
|--------|----------|
| Application | User Services |
| Transport | TCP / UDP |
| Internet | IP Routing |
| Network Access | Physical Communication |
