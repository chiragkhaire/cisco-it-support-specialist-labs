# PowerShell & Bash Administration Handbook

## Overview

PowerShell and Bash are command-line shells used for system administration, automation, and troubleshooting.

This section contains commonly used commands and beginner-friendly administrative scripts with explanations.

The objective is to understand:

- System information gathering
- Process management
- Service management
- Network diagnostics
- User administration
- Storage management
- Basic automation
